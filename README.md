# Related List
It is a custom related list for Account object which is generic for its related list ( Contact, Opportunity, Cases ). 




## Deployment

Get the customRelatedListComponent from the repo and paste it in your sfdx project lwc directory.

```bash
  git clone https://github.com/IamSigma/customRelatedListComponent.git
```
Now simply deploy it to your org.

## Setup

Find customRelatedListComponent and drag it to the account page. Now select the property as shown, or may vary depending on the requirement.
